# JavaScript Practice Projects

A collection of **JavaScript mini-projects** built while learning DOM manipulation, events, user interaction, and browser APIs. Each project focuses on strengthening **core JS fundamentals** through practical, hands-on exercises.

## Key Learnings
- DOM selection and manipulation  
- Handling user events  
- Updating UI dynamically using JS  
- Randomness & game logic  
- Basic Chrome extension development  
- Clean code structure for small apps


## How does each project look?


### [3. Chrome Extension](./3_chrome_extension)
A basic Chrome extension built using HTML, CSS, and JavaScript.  
**Concepts:** Manifest file, DOM scripting inside extension environment, event handling.  
![Chrome Extension Preview](./3_chrome_extension/preview.png)


### [2. Blackjack Game](./2_blackjack_game)
A simple blackjack card game built using JavaScript logic + DOM updates.  
**Concepts:** conditionals, arrays, randomization, game logic, rendering to DOM.  
![Blackjack Game Preview](./2_blackjack_game/image.jpg)


### [1. Passenger Counter](./1_passenger-counter)
A counter app that increments values and saves them in a log.  
**Concepts:** DOM manipulation, event listeners, state updates.  
![Passenger Counter Preview](./1_passenger-counter/station.jpg)
